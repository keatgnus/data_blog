4 설치 (윈도우)

아나콘다 프롬프트를 관리자 권한으로 실행합니다.
VSCode로 진행하시는 분은 VSCode 실행을 관리자권한으로 해야 합니다.
현재 경로를 확인하고 이를 감안하여 적정 경로에 설치 파일을 이동해놓습니다.
