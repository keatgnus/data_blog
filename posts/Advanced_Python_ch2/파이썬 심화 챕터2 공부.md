### Duck typing


```python
# 덕 타이핑: "특정 기능을 지원하는가만 중요하다"는 의미를 전달할 때 사용하는 표현
# = 이터러블 객체
def isiterable(obj):
    try:
        iter(obj)
        return True
    except TypeError: # 이터러블 하지 않은 값
        return False
```


```python
isiterable('a string')
```




    True




```python
isiterable([1,2,3])
```




    True




```python
# 정수는 이터러블 하지 않음
isiterable(5) 
```




    False
